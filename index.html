Blender — Bisect Tool Edit Mode → Bisect → Separate (P)

ADIM 01
📥 STL'i İçe Aktar

Blender'ı aç. Varsayılan küpü sil (X → Delete). Ardından modeli içe aktar.

# Menü yolu:
File → Import → STL (.stl)
# veya OBJ dosyası için:
File → Import → Wavefront (.obj)
💡 Model çok küçük görünüyorsa klavyede Numpad . tuşuna bas — kamerا modele odaklanır.
ADIM 02
✏️ Edit Mode'a Geç

Modele tıkla (seç), ardından Tab tuşuna bas. Edit Mode'a geçersin — mesh noktaları görünür hale gelir.

Tab → Edit Mode'a geç
A → Tüm mesh'i seç (tüm noktalar seçili olmalı)
# Noktalar turuncu görünüyorsa seçili demektir
ADIM 03
✂️ Bisect Tool'u Etkinleştir

Sol paneldeki araç çubuğunda Knife aracını bul. Üzerine uzun tıkla (hold) → açılan menüde Bisect'i seç.

# Sol panelde Knife aracına uzun tıkla →
Bisect seç

# Alternatif kısayol:
Shift + 8 (bazı versiyonlarda)
ℹ️ Bisect aracı, modeli tam düz bir düzlemle keser. Knife'tan farklı olarak kapalı bir yüzey oluşturur.
ADIM 04
📐 Kesim Çizgisini Çiz

3D viewport'ta tıklayıp sürükle — modelin üzerinde bir kesim çizgisi belirir. Bırak.

# Sol alt köşede "Bisect" paneli açılır:
✅ Fill → Kesimleri kapalı yüzey yap (işaretle!)
Clear Inner → Bir tarafı sil
Clear Outer → Diğer tarafı sil
# İkisi de işaretli değilse her iki parça kalır
⚠️ "Fill" seçeneğini mutlaka işaretle! Yoksa kesimdeki yüzey açık kalır, slicer'da sorun çıkar.
ADIM 05
🔀 İki Parçayı Ayır (Separate)

Kesim yapıldı ama parçalar hâlâ aynı objede. Önce bir parçanın mesh'ini seç, sonra ayır.

V → Kesim noktalarını "rip" et (parçaları ayırır)

# Sonra bir parçayı seç:
Ctrl + L → Bağlı mesh'i seç

# Ayrı objeye taşı:
P → "Selection" seç

Tab → Object Mode'a dön
💡 Sağ panelde (Outliner) artık iki ayrı obje görürsün. Her birini ayrı STL olarak export edebilirsin.
ADIM 06
📤 Her Parçayı Ayrı Export Et

Her objeyi ayrı ayrı seç, export ederken Selection Only kutusunu işaretle.

# Bir parçaya tıkla (seç):
File → Export → STL
✅ "Selection Only" kutusunu işaretle
Kaydet → parça_1.stl

# Diğer parça için tekrarla:
parça_2.stl
BONUS
📌 Blender'da Pim Yuvası Ekle (Boolean)

Parçaların hizalanması için pim yuvası eklenebilir. Silindir obje oluşturup Boolean Difference uygularsın.

# Silindir ekle:
Shift + A → Mesh → Cylinder
Boyutunu ayarla (örn: çap 4mm, yükseklik 8mm)

# Boolean Difference (alt parçaya):
Alt parçayı seç → Properties → Modifiers
"+" → Boolean → Operation: Difference
Object: Cylinder → Apply
ℹ️ Üst parçaya silindir ekle (Union), alt parçadan çıkar (Difference) → birbirlerine tam oturur.
⚙️

Blender — Boolean Modifier Union · Difference · Intersect ile şekil birleştirme ve çıkarma

ℹ️ Boolean Modifier nedir? İki 3D objeyi matematiksel olarak birleştirir, çıkarır veya kesişimini alır. Pim yuvası, snap fit, kilit kanalı gibi her türlü bağlantı elemanını bu yöntemle yaparsın.
➕
Union
İki objeyi tek bir objeye birleştirir. Pim çıkıntısı eklemek için kullanılır.
Üst parça + pim
➖
Difference
Bir objeden diğerini çıkarır. Pim yuvası (delik) açmak için kullanılır.
Alt parça − pim = yuva
✖️
Intersect
Sadece iki objenin kesiştiği hacmi bırakır. Karmaşık şekil kesmede kullanılır.
İleri seviye kesimler
ADIM 01
🔵 Silindir (Pim) Objesi Oluştur

Kesim yaptıktan sonra, Object Mode'dayken eklem noktasına bir silindir ekle. Bu silindir hem pim hem de yuva kalıbı olarak kullanılacak.

# Boş alana tıkla (hiçbir şey seçili olmasın)
Shift + A → Mesh → Cylinder

# Silindir boyutunu ayarla (S = Scale):
S → X → 0.004 Enter (4mm çap)
S → Z → 0.008 Enter (8mm uzunluk)

# Konumunu eklem noktasına taşı (G = Grab):
G → Z → değer gir → Enter
💡 Silindirin yarısı üst parçada, yarısı alt parçada olacak şekilde konumlandır. Tam kesim çizgisi üzerinde ortala.
ADIM 02
➕ Üst Parçaya Union Uygula (Pim Çıkıntısı)

Önce üst parçaya silindir ekliyoruz — bu parçada pim dışarı çıkacak.

# 1. Silindiri kopyala (orijinalini koru):
Silindire tıkla → Shift + D → Esc
# (yerinde kopyalandı)

# 2. Üst parçayı seç:
Outliner'da üst parçaya tıkla

# 3. Properties paneli → Modifier Properties (🔧):
"Add Modifier" → Boolean

# 4. Ayarlar:
Operation → Union
Object → Silindir kopyasını seç

# 5. Uygula:
"Apply" butonuna tıkla

# 6. Silindir kopyasını sil (görünmez yap veya sil):
Silindir kopyasına tıkla → H (gizle) veya X (sil)
💡 Üst parçada artık silindir şeklinde bir çıkıntı (pim) var.
ADIM 03
➖ Alt Parçaya Difference Uygula (Yuva/Delik)

Şimdi orijinal silindiri alt parçadan çıkarıyoruz — bu parçada pimin oturacağı yuva açılacak.

# 1. Alt parçayı seç:
Outliner'da alt parçaya tıkla

# 2. Modifier Properties (🔧):
"Add Modifier" → Boolean

# 3. Ayarlar:
Operation → Difference
Object → Orijinal silindiri seç

# 4. Uygula:
"Apply" butonuna tıkla

# 5. Orijinal silindiri sil:
Silindire tıkla → X → Delete
💡 Alt parçada artık pimin tam oturacağı silindirik yuva açıldı.
ADIM 04
📏 Tolerans Ayarı — Kritik!

Pim ve yuva aynı boyuttaysa baskıda birbirine girmez. Yuva için silindiri biraz büyüt.

# Difference işleminden ÖNCE silindiri büyüt:
Silindiri seç → S → 1.05 Enter
# %5 büyüttük → yuva pimden 0.2mm büyük olur

# Tolerans rehberi:
Sıkı geçme (press-fit): +0.1mm
Normal geçme: +0.2mm
Gevşek (kolayca çıkar): +0.3mm
⚠️ Tolerans olmadan pim yuvaya girmez! İlk baskıda +0.2mm ile başla, gerekirse ayarla.
ADIM 05
🔢 Birden Fazla Pim Ekle

Büyük yüzeylerde 2–3 pim olması hem hizalamayı hem de dayanıklılığı artırır.

# Silindiri çoğalt (Array Modifier ile):
Silindiri seç → Modifier → Array
Count: 2 (veya 3)
X offset: eklemler arası mesafe
Apply → sonra Boolean işlemlerini tekrarla

# Veya manuel: Shift+D ile kopyala, konumla
Shift + D → X → mesafe → Enter
ADIM 06
🔍 Sonucu Kontrol Et

Boolean sonrası mesh'i kontrol et — bazı durumlarda N-gon veya hatalı yüzey oluşabilir.

# Edit Mode'da hata kontrolü:
Tab → Edit Mode
Overlay → Statistics'i aç
# Kırmızı kenarlar = sorunlu alan

# Mesh düzeltme (Blender 3D Print Toolbox):
Preferences → Add-ons → "3D Print Toolbox" etkinleştir
N Panel → 3D Print → Check All → Clean Up
💡 Slicer'da "Non-Manifold" hatası alırsan geri dön, Blender'da mesh onar, tekrar export et.

Meshmixer — Plane Cut Edit → Plane Cut → Slice → Separate Shells

ADIM 01
📥 Modeli İçe Aktar

Meshmixer'ı aç. Sol üstteki Import butonuna tıkla, STL veya OBJ dosyasını seç.

Import → STL / OBJ dosyasını seç
# Model ekranda belirir

# Görünümü döndür: Sol tıkla + sürükle
# Zoom: Scroll
# Pan: Orta tık + sürükle
ADIM 02
✂️ Plane Cut Aracını Aç

Sol paneldeki Edit menüsüne tıkla → Plane Cut'ı seç. Ekranda renkli bir düzlem belirir.

Edit → Plane Cut

# Kırmızı/yeşil/mavi oklar: düzlemi taşı
# Küre gizmo: düzlemi döndür
💡 Düzlemi model üzerinde istediğin noktaya sürükle. Eklem (joint) veya doğal sınırları bul.
ADIM 03
⚙️ Cut Type Ayarla: Slice

Sol panelde (Plane Cut menüsünde) Cut Type açılır menüsünü bul.

Cut Type → Slice (Keep Both) seç
# Bu seçenek her iki tarafı da korur

Fill Type → Erase Fill (yüzeyi kapatır)

# Ayarlar tamam →
"Accept" butonuna tıkla
⚠️ "Cut" seçersen bir tarafı siler! "Slice (Keep Both)" seçmeyi unutma.
ADIM 04
🔀 Shell'leri Ayır

Kesim yapıldı ama parçalar hâlâ tek obje. Edit → Separate Shells ile ayır.

Edit → Separate Shells

# Sağda Object Browser açılır:
Object1 (üst parça)
Object2 (alt parça)
# Her biri artık ayrı obje
ADIM 05
📌 Hizalama Pimi Ekle (Silindir Yöntemi)

Prusa'nın önerdiği klasik yöntem: silindir primitif ekleyip Boolean ile pim yuvası oluşturursun.

# 1. Silindir ekle:
Meshmix (sol panel) → Primitives → Cylinder
Modelin kesim yüzeyine sürükle

# 2. Boyutunu ayarla (ok gizmo ile)
Örn: çap ~4mm, uzunluk ~8mm

# 3. Drop Solid menüsünde:
Composition mode → "Create New Object"
Accept

# 4. Silindir'i 3 kopyala (Shift+C)
# 5. Üst parça + silindir → Boolean Union
# 6. Alt parça + silindir → Boolean Difference
💡 Sonuç: Üst parçada pim çıkıntısı, alt parçada yuva. Birbirlerine kilitlenerek oturur.
ADIM 06
📤 Her Parçayı Export Et

Object Browser'da her parçayı sağ tıkla → Export → STL olarak kaydet.

Object Browser → Object1 → sağ tık → Export
→ parça_ust.stl olarak kaydet

Object Browser → Object2 → sağ tık → Export
→ parça_alt.stl olarak kaydet
⚙️

Meshmixer — Boolean İşlemleri Boolean Union · Boolean Difference ile pim ve yuva oluşturma

ℹ️ Meshmixer'da Boolean işlemleri Edit → Boolean menüsü altında. Blender'daki modifier sisteminden farklı olarak burada iki objeyi seçip işlemi tek adımda uygularsın.
ADIM 01
🔵 Pim Silindirini Yerleştir

Kesim tamamlandıktan sonra Object Browser'da parçalar görünür. Şimdi pim olarak kullanacağın silindiri ekle.

# Meshmix panelini aç (sol üst):
Meshmix → Primitives bölümü → Cylinder'ı bul
Kesim yüzeyinin üzerine sürükle ve bırak

# Boyutlandır (gizmo okları ile):
Çap: ~4mm (X ve Y okları)
Uzunluk: ~10mm (Z oku)
# Yarısı üst, yarısı alt parçada olacak şekilde konumla

# Drop Solid menüsünde MUTLAKA:
Composition → Create New Object seç
Accept
⚠️ "Create New Object" seçmezsen silindir parçayla birleşir, Boolean işlemi için ayrı obje olmaz!
ADIM 02
📋 Silindiri Kopyala

Union ve Difference için iki ayrı silindir kopyasına ihtiyacın var — biri pim, biri yuva kalıbı.

# Object Browser'da silindiri seç:
Object Browser → Cylinder'a tıkla

# Kopyala:
Shift + C (veya Edit → Duplicate)
# Artık Cylinder ve Cylinder_copy var

# Tolerans için kopyayı biraz büyüt:
Cylinder_copy'yi seç → Edit → Transform
Scale X ve Y: 1.05 (5% büyük = ~0.2mm tolerans)
💡 Orijinal silindir → Union (üst parçaya pim). Büyütülmüş kopya → Difference (alt parçaya yuva).
ADIM 03
➕ Boolean Union — Üst Parçaya Pim Ekle

Üst parça + orijinal silindir seçilir → Union uygulanır → pim çıkıntısı oluşur.

# 1. Üst parçayı seç:
Object Browser → üst parçaya tıkla

# 2. Silindiri de seç (Ctrl+tık):
Ctrl + tıkla → Cylinder (orijinal)

# 3. Boolean Union uygula:
Edit → Boolean → Union

# 4. Ayarlar penceresinde:
✅ Use Intersection Curves işaretle
✅ Auto-Reduce Result işaretini KALDIR
Target Edge Scale: 5 (daha temiz mesh)
Accept

# Sonuç: üst parçada silindir çıkıntısı (pim) var
ADIM 04
➖ Boolean Difference — Alt Parçaya Yuva Aç

Alt parça + büyütülmüş silindir kopya seçilir → Difference uygulanır → pim yuvası açılır.

# 1. Alt parçayı seç:
Object Browser → alt parçaya tıkla

# 2. Büyütülmüş kopyayı da seç:
Ctrl + tıkla → Cylinder_copy

# 3. Boolean Difference uygula:
Edit → Boolean → Difference

# 4. Ayarlar:
✅ Use Intersection Curves işaretle
✅ Auto-Reduce Result işaretini KALDIR
Accept

# Sonuç: alt parçada silindirik yuva açıldı
💡 Şimdi üst parçanın pimi alt parçanın yuvasına tam oturur. Tolerans sayesinde kolayca girer.
ADIM 05
🔍 Mesh Onar (Gerekirse)

Boolean sonrası mesh'te küçük hatalar oluşabilir. Meshmixer'ın Analysis aracıyla kontrol et.

# Hata kontrolü:
Analysis → Inspector
Kırmızı/mavi noktalar = sorunlu alan

# Otomatik onar:
"Auto Repair All" butonuna tıkla

# Manuel onar:
Sorunlu noktaya tıkla → "Repair" seç
ℹ️ Boolean sonrası Inspector'da hata yoksa doğrudan export edebilirsin.
ADIM 06
📤 Export Et ve Slicer'a Gönder

Her parça ayrı ayrı seçilip STL olarak kaydedilir. Slicer'da toleransı test bas.

# Üst parçayı export:
Object Browser → üst parça seç
File → Export → STL → parça_ust_pimli.stl

# Alt parçayı export:
Object Browser → alt parça seç
File → Export → STL → parça_alt_yuva.stl

# Slicer'da tolerans testi:
Önce sadece eklem bölgesini küçük ölçekte bas
Pim yuvaya giriyorsa → tam ölçek baskıya geç
⚠️ Büyük figürü basmadan önce toleransı test et! Aksi halde saatlik baskı çöpe gidebilir.

Bağlantı Tipleri Hangi connector ne zaman kullanılır?

🔌
Plug / Dowel
Silindirik pim. Bir tarafa çıkıntı, diğerine delik. Yapıştırma gerektirmez ama gevşeyebilir.
Kol, bacak eklemleri
🔗
Snap Fit
Çıt-çıt sistemi. Videodaki figürde bu kullanılıyor. Kilitlenir, sökülür.
Tekrar sökülen parçalar
🪤
Dovetail
Kırlangıç kuyruğu. Hem hizalar hem tutar. Gövde-kaide bağlantısı için ideal.
Düz yüzeyler / kaide
ℹ️ Bambu Studio / Orca Slicer'da bunları otomatik ekleyebilirsin: Model seç → C → Cut Tool → "Add Connectors" → tip seç. Blender veya Meshmixer'a gerek kalmaz.

Spray Boyama için İpuçları Parçalı baskıyı boyamaya hazırlarken dikkat et

📍 Kesim Noktasını Akıllı Seç

Doğal eklem noktalarından kes: omuz, diz, bel, boyun. Bu noktalardaki kesim çizgisi montaj sonrası görünmez. Düz yüzeylerden kes — eğri kesimler daha zor yapışır.

🎨 Boya Önce, Yapıştır Sonra

Her parçayı ayrı ayrı spray ile boya. Kuruttuktan sonra birleştir. Böylece eklem kıvrımlarına kadar boya ulaşır — monte edilmiş halde boyarsan o kısımlar ham kalır.

📏 Tolerans Bırak

Pim/yuva oluştururken slicer'dan önce 0.2–0.3mm tolerans ekle. Aksi takdirde parçalar birbirine oturmaz. Bambu Studio'da connector eklerken "Tolerance" değerini 0.2 olarak ayarla.

🔧 Önce Test Et

Büyük figürü basmadan önce sadece eklem bölgesini küçük ölçekte test bas. Pim/yuva uyumu iyi mi kontrol et, gerekirse toleransı ayarla, sonra tam ölçekte bas.

🚀 Tavsiye: Başlangıç için Orca Slicer

Blender ve Meshmixer daha fazla kontrol sunar ama öğrenme eğrisi var. Hızlı başlamak için Orca Slicer'ın dahili Cut Tool'u (C tuşu) kullan — connector ekleme dahil her şey tek arayüzde.
